## 4. 환경 변수 설정 (선택사항)

`.env.local` 파일이 이미 포함되어 있으므로 추가 설정 불필요.
단, D 드라이브 경로가 다르다면 수정:

```env
ANDROID_SDK_ROOT=D:/CLAUDE_CLI/Android_SDK
ANDROID_HOME=D:/CLAUDE_CLI/Android_SDK
EXPO_CACHE_HOME=D:/CLAUDE_CLI/expo_cache
ANDROID_AVD_HOME=D:/CLAUDE_CLI/android_avd
```

## 5. 빌드 및 실행

### Development Build (현재 설치된 APK용)
```bash
npx expo start --dev-client
```

### Preview Build (독립 실행형 APK)
```bash
npx eas build --profile preview --platform android
```

### Production Build
```bash
npx eas build --profile production --platform android
```

## 8. 문제 해결

### "Project archive too big" 에러
- **원인**: ColorPlayExpo가 독립 Git 레포가 아님
- **해결**: 이미 독립 레포로 설정되어 있으므로 문제없음

### 포트 충돌 (Port 8081 already in use)
```bash
npx kill-port 8081
```

### 캐시 문제
```bash
npx expo start --clear
```
